# 🧫 セル・オートマトン実験室（Life Lab）

Python + uv + NiceGUI を使って作った、ブラウザで動くライフゲーム（セル・オートマトン）実験アプリです。
グリッドをクリックして細胞を配置し、世代の進化をリアルタイムに観察できます。

---

## ✨ 特長

- ✅ ブラウザで動作（インストール不要）
- ✅ クリックでセルの配置・削除が可能
- ✅ スタート／ストップ切り替え
- ✅ 世代更新のスピード調整
- ✅ 1ステップ実行
- ✅ Python 単体で完結
- ✅ uv による依存管理

---

## 🖥 デモ画面

※ ローカルで起動してブラウザで確認してください。

---

## 🛠 必要環境

- Python 3.9 以上
- uv（Python パッケージマネージャ）
- ブラウザ（Chrome, Firefox, Edgeなど）

---

## 📦 インストール

### uv が入っていない場合

```bash
curl -Ls https://astral.sh/uv/install.sh | sh
````

---

## 🚀 起動方法

```bash
git clone https://github.com/nagata-minoru/life_lab
cd life_lab

uv add nicegui
uv run python main.py
```

起動後、ブラウザで以下にアクセス：

```
http://localhost:8080
```

---

## 🧪 テストの実行

開発用依存を入れて pytest を実行します。

```bash
uv sync --group dev
# パッケージ解決のためモジュール形式で実行
uv run python -m pytest
```

## 🎮 使い方

### 🟩 セルの操作

* セルをクリック → 生 / 死 を切り替え

### ▶ スタート

* 自動で世代が進みます

### ⏸ ストップ

* 進化を一時停止

### ⏭ 1ステップ

* 1世代だけ進めます

### ⏱ スピード調整

* スライダーで世代更新間隔を変更

### 🧹 クリア

* すべてのセルをリセット

### 🎲 初期パターン再配置

* 停止中のみボタンが有効になり、R-ペントミノと3方向グライダーを再配置

---

## 📐 ルール（Conway's Game of Life）

生存:

* 周囲の生きたセルが 2 または 3 → 生存

誕生:

* 周囲の生きたセルが 3 → 誕生

死滅:

* それ以外は死

盤面はトーラス構造（上下左右がつながる）として処理しています。
